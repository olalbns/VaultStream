Comment ça fonctionne

StreamVault est un lecteur vidéo universel avec serveur proxy local. Voici tout ce qu'il peut faire.

🚀

Démarrage rapide

  1. Installe Python 3.8+ et yt-dlp : pip install yt-dlp
  2. Lance le serveur : python server.py
  3. Ouvre http://localhost:5000
  4. Colle n'importe quel lien vidéo et clique Lire
🔄

Cascade de lecture

Pour chaque lien, StreamVault essaie automatiquement :

1
API du siteAppelle directement l'API hakunaymatata pour obtenir les URLs signées fraîches
2
yt-dlpRésout les liens CDN signés de 1000+ sites comme IDM
3
Proxy PythonRelaie les requêtes côté serveur pour contourner le CORS
4
Blob proxyTélécharge en mémoire puis lit localement
5
NatifLecture directe <video src> en dernier recours
📡

Formats supportés

MP4WebM HLS / M3U8DASH / MPD YouTubeVimeo CDN signéJSON API MKVMOV

Pour les playlists, colle l'URL de la playlist YouTube ou d'une chaîne.

📋

Playlists

  1. Colle l'URL d'une playlist ou chaîne YouTube
  2. StreamVault détecte automatiquement et extrait toutes les vidéos
  3. Le panneau latéral affiche la liste — clique sur une vidéo pour la lire
  4. La lecture s'enchaîne automatiquement
  5. Télécharge toute la playlist ou une sélection depuis la page Télécharger
📋

JSON direct

Si le site expose une API JSON avec les URLs (ex: hakunaymatata), colle le JSON directement dans le champ :

{"data":{"downloads":[
  {"url":"https://cdn.../video.mp4",
   "resolution":1080,
   "size":"500000000"}
]}}

StreamVault extrait les URLs, les trie par qualité et affiche le sélecteur.

↓

Téléchargement

  1. Clique ↓ Télécharger sous n'importe quelle vidéo en lecture
  2. Ou va dans Télécharger et analyse une URL
  3. Choisis le format, la qualité, la piste audio, les sous-titres
  4. La barre de progression s'affiche sous la vidéo
  5. Quand c'est prêt, clique Sauvegarder pour télécharger
🔌

Extension navigateur (IDM-like)

  1. Ouvre chrome://extensions
  2. Active le Mode développeur
  3. Clique Charger l'extension non empaquetée
  4. Sélectionne le dossier streamvault/extension/
  5. L'extension intercepte les requêtes vidéo exactement comme IDM
  6. Elle envoie les headers (cookies, tokens) à StreamVault automatiquement
📁

Collections & Queue

Queue : liste temporaire de lecture. Ajoute des vidéos depuis n'importe où avec + Queue. La lecture s'enchaîne.

Collections : dossiers persistants. Crée une collection, donne-lui un nom, une couleur, une icône. Ajoute des vidéos individuellement ou une playlist entière en un clic. Elles sont sauvegardées sur disque.

⚙

API du serveur

GET /api/resolve?url=Résout une URL via API site / yt-dlp
GET /api/proxy?url=Stream relay avec support Range (seek)
GET /api/playlist?url=Extrait toutes les vidéos d'une playlist
GET /api/video/info?url=Formats, audio, sous-titres d'une vidéo
POST /api/ytdl/downloadLance un téléchargement yt-dlp
POST /api/ytdl/download/batchTéléchargement en lot (playlist)
GET /api/collectionsListe les collections
POST /api/interceptReçoit les URLs depuis l'extension
